Detailed Description

Provides facilities for a counter by components with limited sizes.

Given a component counter with sizes (3, 2):

sizes[0] = 3;
sizes[1] = 2;

a ComponentCounter counts from (0, 0) to (2, 1) where each increment looks like:

(0, 0)
(1, 0)
(2, 0)
(0, 1)
(1, 1)
(2, 1)
